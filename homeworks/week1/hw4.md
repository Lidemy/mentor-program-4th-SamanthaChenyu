## 跟你朋友介紹 Git

1. 將你變動的檔案加進 git 索引裡，使用 `git add`
2. 你不確定你現在有變動哪些檔案的話可以使用 `git status` 查看檔案的狀態
3. 每當你對你的版本進行變動，都會需要下一個 `git commit -m"`對這個版本的變動大致說明"來提交版本
4. `git push` 就是把檔案推上去遠端數據庫
5. 如果今天有路人覺得你的笑話好笑，他就可以透過 `git clone` 複製你的笑話到他的本端
6. 然後他覺得他也有新的想法想要回饋你，他可以在你的 repo 裡點選 fork 專案到他自己的帳號底下
7. 改完後，他就使用 push 先推到自己的帳號專案，然後他發個通知 pull request ，請你來看一下
8. 你看完後，覺得很好。就可以使用 merge 把路人做的修改合併到你的專案裡。
